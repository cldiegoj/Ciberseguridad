Copiamos el exploit en la carpeta para su uso: 

searchsploit -m linux/remote/45939.py

El script, usa python2, requiriendo 2 argumentos (IP, usuario)

Con el script podemos enumerar usuarios

[+] root is a valid username
[-] admin is an invalid username
[-] user is an invalid username

Quedandonos con root como usuario valido

Pero falta la password, por lo que usaremos hydra:
